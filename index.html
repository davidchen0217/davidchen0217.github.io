Visual Studio Code for Java & JavaScript

發表於 2019-09-11

1.下載Visual Studio Code 安裝檔 :https://code.visualstudio.com/download

逐步安裝完，會有Visual Studio Code IDE 和 Java Extension Pack. 包含

Language Support for Java(TM) by Red Hat
Debugger for Java
Java Test Runner
Maven for Java
Java Dependency Viewer

2.需另安裝Maven，並設定環境變數 : 參考 https://www.dev2qa.com/create-java-project-with-maven/

3.第2步驟安裝完成，即可使用 Maven for Java 對專案執行 clean package -DskipTests 編譯出 war 檔 或 jar 檔

war檔有兩個地方要注意

3.1. src\main\webapp\META-INF\ 下需包含context.xml

3.2. src\main\config\dev\db.properties 的Oracle資料庫連線設置

4.安裝 Tomcat for Java 並指定Tomcat Server的安裝目錄後，可對war檔右鍵->Run on Tomcat Server將會開始佈署至Tomcat Server.

5.SVN control

5.1 安裝小烏龜作爲基底，以及第一次取得專案

5.2 Interface

5.2.1 cmd-line mode: 安裝 TortoiseSVN 即可對專案右鍵或下command進行SVN 操作.

5.2.2 Integrated mode: SVN

6.匯入Java Formatter.xml，匯出Eclipse的formatter.xml，將檔名與配置檔案名稱設定成一樣，在下面這頁設定即可

7.靜態程式碼檢查，安裝Checkstyle for Java該連結提供兩種xml格式

將xml檔案點選右鍵 -> Set Checkstyle Configuration File 即可設定完成

8.JavaScript Editor

8.1 安裝 Eslint

9.SQL Query

9.1 安裝 Oracle Developer Tools for VS Code

9.2 查詢資料庫：完成設定資料庫 –> 下命令 Oracle: Develop New Query –> 即可查詢

10.安裝 Apex PMD

11.HotKey

11.1 keyboard-shortcuts-linux.pdf

11.2 keyboard-shortcuts-macos.pdf

11.3 keyboard-shortcuts-windows.pdf

12.JSP Editor

12.1 Java Server Pages (JSP)

已知缺點

JSP 支援薄弱
Java Resource Bundle 不支援

Reference:

官方文件:

Writing Java with Visual Studio Code

Java Project Management in VS Code

用Hexo建立Blog

發表於 2019-09-11

  1. Hexo事前安裝準備:node.js + Git。

  2. 登入GitHub帳號並新增一個repository且Repository name 必須命名為 UserName + “.github.io”。

  3. 使用Git Bash開啟指令視窗並輸入以下指令來獲得ssh-rsa key。

    3.1接下來詢問都按Enter略過

  4. 從ssh key文件預設位置 C:\Users\用戶名.ssh下，找到id_rsa.pub文件(文件內全都是key的一部分)。

  5. 在GitHub -> Setting -> SSH and GPG keys，新增SSH keys
    5.1 Title任意取即可。
    5.2 key填入id_rsa.pub文件內所有內容。

  6. 確認node.js 是否安裝成功

  7. 新增hexo blog資料夾，cmd -> cd至資料夾 -> 安裝hexo

  8. 初始化blog

  9. cd 至資料夾\blog

  10. 新增新文章
    10.1 hexo new my_first_post

  11. 產生靜態檔案
    啟動local server , default port 4000

    更換port

12.在\blog下找到配置文件_config.yml
13.1 修改deploy配置

13.部署至GitHub

13.1 hexo d (這步如有出現ERROR Deployer not found: git錯誤),在\blog下安裝 npm install hexo-deployer-git –save
14.在browser輸入 “UserName”.github.io 即可看到Blog
